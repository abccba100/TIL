## 1. Java 변수 선언


### 1.1 Java 변수 선언 방법

Java에서는 C랑 변수 선언 방법이 똑같다.

```java
자료형 변수 이름

ex) int age;
ex) String name;
```

### 1.2 Java 자료형

| **자료형** | **설명** | **예시** |
| --- | --- | --- |
| **int** | 정수 | **10, -5** |
| **long** | 엄청 큰 수 담을 때 사용 숫자 뒤에 `l` 이나 `L` 붙어야 정상적으로 인식 | **3L, 10000000000L** |
| **byte** | 1바이트 -128 ~ 127, 메모리 아낄 때 | **-128, 0, 1, 127** |
| **double** | 실수, 정밀도 높은데 메모리 많이 듬 | **3.14, -0.001** |
| **float** | 실수, 정밀도 낮은데 메모리 적게 듬 숫자 뒤에 `f` 나 `F` 붙어야 정상적으로 인식 | **3.14f, 0.01f** |
| **char** | 단일 문자, 작음 따옴표로 감쌈 | **'A', '가'** |
| **boolean** | true나 false | **true, false** |

### 1.3 Java 값 할당

다른 언어랑 마찬가지로 =를 이용해서 값을 할당한다.

```java
int age = 3;
```
