## 1. Spring


### 1.1 Spring 등장

다양한 언어를 기반으로한 프레임워크가 등장하던 시절,
Spring이라는 JAVA 언어를 기반으로 웹 서버를 쉽게 만들 수 있게 해주는 프레임워크가 등장했다

## 2. Spring Boot


### 2.1 Spring Boot 등장

2000년대 초반에 JAVA 기반 애플리케이션은 개발은 복잡한 설정, 무거운 서버 등 상당히 어려운 작업이었다.
개발자들이 기능 개발을 하는 시간보다 설정을 작업하는데 더 많은 시간을 쏟을 정도였다.

이때 등장한 게 Spring Boot로 개발자들이 자주 사용하는 패턴들을 미리 프로젝트에 적용시켜 두어
개발자들이 개발을 훨씬 빠르게할 수 있게 해줬다.
