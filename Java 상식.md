## 1. Java 상식

### 1.1 JVM

java 언어로 쓰인 코드를 기계어로 번역할 때
java → 바이트 코드 → 기계어 순으로 번역한다.

이때 바이트 코드에서 기계어로 번역해주는 역할을 하는 게 JVM이다

### 1.2 JRE

Java에서 자주 사용되는 라이브러리 같은 도구들이 담겨 있는 걸로 JVM도 JRE 안에 담겨 있고,
System.out.println() 같은 기능을 별도의 설정 없이 사용할 수 있는 것도 JRE 라는 실행 환경 덕분이다.

### 1.3 JDK

JDK도 자바 애플리케이션(Java 프로그램)을 개발할 때 필요한 도구들이 담겨 있는 것이다.
하지만 JRE 보다 더 큰 개념으로 JDK 안에 JRE가 들어있고, java → 바이트 코드로 해석해주는
컴파일러(javac)도 JDK 안에 들어있다.
