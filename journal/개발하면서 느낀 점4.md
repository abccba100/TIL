# 📝 Raw 기록

## **상황**

컴포넌트를 새롭게 개발하고 수정하면서 stories 파일과 test 파일을 생성하고 코드를 작성해야 하는 상황

## **내가 한 일**

어차피 stories 파일과 test 파일을 중요한 로직이 포함되어 있지 않고 코드의 구조가 반복된다고 생각하여  
생성형 AI를 통해 빠른 컴포넌트 개발을 함

## **결과**

기존에는 stories 파일과 test 파일 작성에 많은 시간을 보내야 했지만 이제는 생성형 AI에게 해당 테스크를  
맡기고 나는 결과를 확인, 검토하면 된다

두 파일에는 중요한 로직이 들어간다고 생각하지 않아 AI 사용에도 부담이 되지 않았고 개발 속도를 빠르게  
올린 것 같아 기억에 남는다 앞으로는 이런 것처럼 중요하지 않고, 오래 걸리며, AI가 잘 할 수 있는 테스크가  
뭔지 파악하고 그걸 AI를 통해 빠르게 테스크를 해결하는 것이 점점 중요해지지 않을까 싶다
